What is a Docker Image?

A Docker Image is a ready-made package that contains:

Application code
Runtime (like Python, Node, .NET)
Libraries & dependencies
OS-level tools

👉 In simple terms:
Docker Image = Blueprint / Template

==================docker process flow================
--Build Docker Image
docker build -t fastapi-app .

--Verify Image
docker images

--Start Kubernetes (Docker Desktop)
👉 Enable Kubernetes in Docker Desktop (one-time)

--Check Cluster
kubectl get services
kubectl get nodes
kubectl logs -l app=fastapi-app

--Create Deployment
kubectl create deployment fastapi-app --image=fastapi-app:latest

--Fix Local Image Issue(if required)
kubectl edit deployment fastapi-app
</> YAML code change
imagePullPolicy: Never

--Expose Service
kubectl expose deployment fastapi-app --type=NodePort --port=8000

--Get Service Port
kubectl get services

--Open in Browser
http://localhost:31374

--Alternative (always works)
kubectl port-forward service/fastapi-app 8001:8000

--browser open
http://localhost:8001/docs

--Final Flow
Docker build
→ Kubernetes deployment
→ Service expose
→ Browser access ✅
